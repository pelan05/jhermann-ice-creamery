# Overview

These are recipes from outside sources or contributed by users.

!!! info "Pull Request Guideline"

    If you want to contribute via a pull request, make sure you paste your recipe
    into [this notebook](https://notebooklm.google.com/notebook/4849972b-b449-40b9-bc93-5a0660ef518d/preview)
    to get it properly formatted.

    Also check that the links to the food database are mapped right to make the nutrition label correct.
    If the AI looked up the wrong food, do it manually at [AFCD](https://afcd.foodstandards.gov.au/)
    and put something like `use F008065 for the red miso` into the prompt (mapping to soy sauce,
    with a similar nutrition profile).

    Also mention the source, so link to your GitHub account for example.

    The notebook helps you formulate new recipes, or format an ingredient list you already have.

    - **To Create:** "Give me a high-protein Chocolate Peanut Butter recipe for my Deluxe."
    - **To Format:** "Here is my messy recipe notes. Can you format this for me? ..."
